## Quick Start

### Prerequisites
- Node.js (v18+)

### Setup
```bash
npm install
npm run dev
```

### Build
```bash
npm run build
npm start
```

## Scripts

- `npm run dev` - Development server
- `npm run build` - Production build
- `npm start` - Production server
- `npm run lint` - Run ESLint
